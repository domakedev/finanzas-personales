# Estructura de Logos

Coloca los logos de bancos y billeteras en esta carpeta con los siguientes nombres:

## Bancos
- `interbank.png` - Logo de Interbank
- `bcp.png` - Logo de BCP
- `bbva.png` - Logo de BBVA

## Billeteras Digitales
- `yape.png` - Logo de Yape
- `plin.png` - Logo de Plin

**Recomendaciones:**
- Formato: PNG con fondo transparente
- Tamaño: 200x200px o superior (se ajustará automáticamente)
- Peso: Menos de 50KB por logo
